# x-koa

框架目录结构
>
├── app.js 				应用入口
├── config				配置文件
│   ├── default.json
│   ├── develop.json
│   └── production.json
├── dist				官方站点
│   ├── index.html
│   └── static
├── node_modules		系统模块
├── package.json
├── src
│   ├── auth			认证模块
│   ├── controller		控制器
│   ├── model			数据模型
│   ├── nodebatis		Dao层框架
│   ├── sequelize		Dao层框架
│   └── yaml			SQL文件
└── static				静态资源
    └── scanlogin.html
